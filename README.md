# React Router + TypeScript Boilerplate

This is a minimal Vite-based React + TypeScript starter with React Router v6.

## Quick start

1. Install dependencies all dependencies npm install
   ```bash
   npm install
   ```

2. Run dev server
   ```bash
   npm run dev
   ```

3. Build
   ```bash
   npm run build
   ```

## Create a GitHub repo and push

```bash
git init
git add .
git commit -m "Initial boilerplate"
# create repository on GitHub (web or use `gh` CLI), then:
git remote add origin git@github.com:<your-username>/<repo-name>.git
git branch -M main
git push -u origin main
```
